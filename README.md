# Rust Sandbox

> https://doc.rust-lang.org/rust-by-example/

## Install

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

```bash
cargo --version
rustup --version
```

## Usage

1. Start new project
```bash
cargo new $projectName
```

2. Compile
```bash
cargo build
```

3. Run 
```bash
cargo run
```

4. Test
```bash
cargo test
```
